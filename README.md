# Blog - Plataforma de Artigos

Blog simples para criar e publicar artigos. Este projeto utiliza **PHP** para o backend e **MySQL** para o banco de dados. Permite aos usuários criar, editar e listar posts.

## Funcionalidades
- **Publicar artigos:** Crie e publique novos posts com título e conteúdo.
- **Editar artigos:** Modifique posts já publicados.
- **Listar artigos:** Exiba todos os artigos criados.

## Tecnologias Utilizadas
- **PHP** - Backend para gerenciamento dos artigos.
- **MySQL** - Armazenamento dos artigos no banco de dados.
- **HTML5** e **CSS3** - Interface de usuário.
- **JavaScript** - Interatividade e validação.
